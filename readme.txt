------------------------------CET_TRANSLATOR-------------------------

Translate your MOD text's to the desired Cyberpunk2077 Language .
This mod is a tool for CET modders that translate String Variables .

-------------------------------Instalation----------------------------

Install Php on your local machine
https://www.php.net/
Extract the CET_TRANSLATOR to you CyberEngineTweeks Directory
Cyberpunk2077\bin\x64\plugins\cyber_engine_tweaks

-----------------------------------Usage------------------------------
Config set.lua 
Write  text , sourceLanguage and the outputfilename
compile set.lua to store your translations on the outputfile
init.lua have built-in function to get the translated text, at the selected Cyberpunk UI Language
