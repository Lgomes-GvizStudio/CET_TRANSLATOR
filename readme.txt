------------------------------CET_TRANSLATOR-------------------------

Translate your MOD text's to the desired Cyberpunk2077 Language .

-------------------------------Instalation----------------------------

Install Php on your local machine
https://www.php.net/
Extract the CET_TRANSLATOR to you CyberEngineTweeks Directory
Cyberpunk2077\bin\x64\plugins\cyber_engine_tweaks

-----------------------------------Usage------------------------------
Config set.lua 
Write  text , sourceLanguage and the CET_TRANSLATOR outputfilename
compile set.lua (before load your MOD)
init.lua have built-in function to get the translated text
